## ✨ Features

- 🎨 **Dark purple & black theme** — data science aesthetic
- 🌌 **Animated aurora background** — live canvas animation
- ⌨️ **Typing animation** in hero section
- 📊 **Animated skill proficiency bars** — scroll triggered
- 🃏 **Glassmorphism project cards** with hover glow effects
- 📱 **Fully responsive** — works on mobile, tablet, desktop
- ⚡ **Scroll progress bar** at the top
- 🔮 **Floating particle dots** in hero
- 🔍 **SEO optimized** — meta tags, OG tags, schema markup
- 🚀 **Deployed on GitHub Pages** — zero cost hosting
